This is a Stunning AI Landing Page: Next.js, Tailwind, and CSS Animation.

- **Next.js**
- **TailwindCSS**
- **Framer Motion**


## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```
